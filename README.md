
# Finkel

Save your soul and leave while you still have time!

## Build

    make build

### Prerequisites

+ Server
  1. Thrift 0.7.0
  2. g++
+ Client
  1. Thrift
  2. node.js

## Run

1. Compile `FinkelServer.cpp` on a Windows machine.
2. Run `FinkelServer.cpp`.
3. Set the host and port of the thrift service in `client/bin/bot.js`.
4. `client/bin/bot.js`
